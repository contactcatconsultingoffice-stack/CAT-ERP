Prompt de Configuration : Système de Gestion Interne (CRM) - CAT ConsultingContexte du projet :Développer une plateforme de gestion interne (ERP/CRM) pour CAT Consulting, un cabinet de conseil spécialisé en stratégie, innovation et développement web. L'application doit permettre de gérer le cycle de vie complet des projets, de la prospection à la facturation finale.Stack Technique Imposée :Frontend : Next.js (App Router) avec Tailwind CSS et les composants shadcn/ui.Backend : Node.js avec une architecture API REST/Server Actions.Base de données : PostgreSQL avec Prisma ORM (pour la rigueur des données financières).Authentification : NextAuth.js avec gestion des rôles : ADMIN (CEO - Cheick Ahmed Thiam) et COLLABORATOR (Experts/Freelances).Architecture des Données (Schéma Prisma) :Créer un schéma de base de données incluant les relations suivantes :User/Collaborator : ID, nom, rôle, expertise (Design, Dev, Stratégie), et un champ de suivi des actions de promotion sur les réseaux sociaux.Partner : Entreprises partenaires (ex: Nova Prime Management) avec suivi des accords de co-traitance.
3. Project : Lié à un client, incluant le type (Portfolio, Blog, E-commerce, Application) et le statut.
4. Financial (Invoice/Quote) : ID, montant (HT/TTC), échéancier (50% signature / 50% livraison), statut du paiement (En attente, Payé, Retard).Fonctionnalités Clés à Générer :Dashboard Collaborateur : Interface permettant de soumettre une fiche de suivi de mission (tâches réalisées, heures, actions de promotion effectuées).Générateur de Documents : Intégrer un module (Puppeteer/React-pdf) pour générer des devis et contrats PDF à partir des templates de CAT Consulting (incluant les clauses de propriété intellectuelle et de confidentialité).Suivi Financier : Vue d'ensemble des revenus, des factures en attente et des paiements via Lemon Squeezy ou Stripe.Gestion d'Équipe : Annuaire des experts internes et de leurs disponibilités.Instructions de Design :Interface moderne, épurée, type SaaS professionnel.Utiliser Framer Motion pour des transitions fluides entre les pages du dashboard.Mode sombre/clair supporté.
